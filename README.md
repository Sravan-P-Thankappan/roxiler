# Task
Api Development



## Tech Stack

**Server:** Node, Express.js, Mongoose




## Run Locally

Clone the project

```bash
  https://github.com/Sravan-P-Thankappan/roxiler_task.git
```

Go to the project directory

Install dependencies

```bash
  npm install
```

Start the server

```bash
 npm start

```

Mongodb connection

```bash
mongodb://localhost:27017

```
Database Name
```bash
  roxiler
```
